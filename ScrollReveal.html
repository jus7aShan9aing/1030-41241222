ScrollReveal 圖片動畫

ScrollReveal 官網

範例程式碼


            <script src="https://unpkg.com/scrollreveal"></script>

            ScrollReveal().reveal('.box', { 
            duration: 1000,      // 動畫持續時間
            distance: '50px',    // 起始位置
            origin: 'bottom',    // 從下方向上出現
            reset: true          // 滾動出視口後，元素會重置動畫
            });
